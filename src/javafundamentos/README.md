# javafundamentos

Aplicação Java de console que identifica texto, datas, data e hora, inteiros,
decimais, booleanos e arrays simples. Ela também extrai os dígitos de entradas
de telefone e os formata nos padrões abaixo:

- Fixo: `xxxx-xxxx` ou `(xx)xxxx-xxxx`
- Celular: `xxxxx-xxxx` ou `(xx)xxxxx-xxxx`

## Executar

Na pasta `javafundamentos`:

```bash
javac -d out src/javafundamentos/*.java
java -cp out javafundamentos.Main
```

Digite `sair` para encerrar a aplicação.
