# ☕ Bootcamp NTT DATA: Backend Java com Spring AI

Repositório dedicado aos estudos, exercícios e projetos desenvolvidos durante o **Bootcamp NTT DATA: Backend Java com Spring AI**, promovido pela **DIO (Digital Innovation One)** em parceria com a **NTT DATA**.

## 📚 Sobre o Bootcamp

Este bootcamp tem como objetivo formar desenvolvedores Back-end utilizando **Java** e o ecossistema **Spring**, abordando desde os fundamentos da linguagem até a construção de aplicações modernas com integração de Inteligência Artificial.

Ao longo da jornada são explorados conceitos essenciais para o desenvolvimento de sistemas robustos, escaláveis e seguros, seguindo boas práticas de mercado.

**Carga horária:** 54 horas  
**Nível:** Intermediário

---

## 🚀 Conteúdos abordados

- Fundamentos da linguagem Java
- Programação Orientada a Objetos (POO)
- Herança e Polimorfismo
- Tratamento de Exceções
- Collections e Streams
- APIs REST com Spring Boot
- Spring Security
- Clean Architecture
- OpenFeign
- Integração com APIs externas
- Inteligência Artificial com OpenAI Whisper
- Inteligência Artificial com Google Gemini
- Desenvolvimento de um Assistente Virtual por Voz

## 🎯 Objetivos

- Consolidar os fundamentos da linguagem Java.
- Praticar Programação Orientada a Objetos.
- Desenvolver APIs REST utilizando Spring Boot.
- Aplicar boas práticas de arquitetura e organização de código.
- Integrar aplicações com serviços externos.
- Explorar recursos de Inteligência Artificial no desenvolvimento Back-end.
- Construir projetos que possam compor meu portfólio.
